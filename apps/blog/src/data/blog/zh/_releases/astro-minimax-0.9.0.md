astro-minimax 0.9.0 是一个功能增强版本，带来了多项用户体验改进和组件增强。

## 新功能

### Mermaid 交互工具栏

增强了 MermaidInit 组件，新增交互式工具栏：

- **缩放控制**：放大、缩小、重置缩放
- **导出功能**：支持导出为 SVG 和 PNG 格式
- **全屏模式**：全屏查看图表
- **主题切换**：在图表中快速切换主题
- **改进渲染**：优化了图表渲染性能和视觉效果

### 增强的评论组件

优化了 Comments 组件的功能和用户体验：

- **加载超时**：新增加载超时机制，避免无限等待
- **错误处理**：完善的错误处理和友好的错误提示
- **重试机制**：加载失败时提供重试选项
- **状态反馈**：清晰的加载状态和进度指示

### UI 组件改进

增强了多个 UI 组件的样式和可访问性：

- **改进的按钮样式**：更清晰的交互反馈
- **增强的可访问性**：更好的键盘导航和屏幕阅读器支持
- **优化的动画效果**：更流畅的过渡动画
- **统一的设计语言**：一致的颜色和间距

## 改进

### 用户偏好设置

增强了用户偏好设置系统：

- **默认配置**：提供合理的默认配置
- **设置集成**：与设置面板更好地集成
- **存储优化**：优化了偏好设置的存储和读取

### 文档增强

- **AI 指南更新**：更新了 AI 功能指南，包含新功能和配置说明
- **CLI 文档**：完善了 CLI 工具的文档和使用示例

### OG 图片生成

- **工具更新**：更新了 OG 图片生成工具
- **截图配置**：增强了截图相关配置选项

## 修复

- **剪贴板复制反馈**：修复了按钮 innerHTML 更新问题，提供更清晰的复制成功反馈
- **网站 URL 更正**：修复了配置中错误的网站 URL

## 升级指南

如果你使用的是 0.8.3，只需更新依赖：

```bash
pnpm update @astro-minimax/core @astro-minimax/ai @astro-minimax/cli @astro-minimax/notify
```
