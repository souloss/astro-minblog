astro-minimax 引入了全新的设置面板，让你可以个性化博客的外观和体验。

## 打开设置面板

点击页面右下角的设置图标（齿轮图标）即可打开设置面板。

## 设置选项

### 外观设置

| 选项 | 说明 | 可选值 |
|------|------|--------|
| **配色方案** | 主题配色 | teal、ocean、rose、forest、midnight、sunset、mono、github |
| **边框圆角** | 组件圆角大小 | 小、中、大、超大 |
| **字体大小** | 全局字体缩放 | 0.85x - 1.25x |

### 阅读设置

| 选项 | 说明 | 可选值 |
|------|------|--------|
| **阅读模式** | 开启后进入更专注的阅读体验 | 开 / 关 |
| **字号** | 文章正文字号 | 小、中、大、超大 |
| **行高** | 文章正文行高 | 紧凑、舒适、宽松 |
| **内容宽度** | 文章内容区域宽度 | 窄、中、宽 |
| **阅读主题** | 文章阅读主题 | 默认、护眼、羊皮纸、夜间、OLED |
| **字体** | 文章正文字体 | 衬线、无衬线、等宽、系统、代码、霞鹜文楷、站酷字体、Readable、Classic |
| **专注模式** | 突出当前阅读段落 | 开 / 关 |

### 布局设置

| 选项 | 说明 | 可选值 |
|------|------|--------|
| **文章列表布局** | 首页文章列表显示方式 | 卡片（标题优先）、网格（三列快速浏览）、列表（紧凑归档） |

### 通用设置

| 选项 | 说明 |
|------|------|
| **吸附返回顶部** | 显示/隐藏文章页内的吸附式返回顶部按钮 |
| **返回顶部** | 显示/隐藏返回顶部按钮 |
| **主题切换** | 显示/隐藏主题切换按钮 |
| **阅读时长** | 显示/隐藏文章阅读时长 |
| **动画效果** | 开启/关闭页面动画 |
| **卡片悬停** | 开启/关闭卡片悬停效果 |
| **平滑滚动** | 开启/关闭平滑滚动 |

### AI 聊天面板尺寸

启用 AI 聊天时，可调整对话面板宽度：

| 选项 | 说明 |
|------|------|
| **小（S）** | 紧凑宽度，适合边浏览边对话 |
| **中（M）** | 默认平衡尺寸 |
| **大（L）** | 更宽面板，适合长回答与代码阅读 |

所选尺寸会保存到浏览器 `localStorage`，下次访问自动恢复。

## 分享设置

你可以将当前的设置分享给他人：

1. 点击设置面板底部的「分享」按钮
2. 当前偏好会被编码并复制到剪贴板，用于生成可分享链接
3. 他人打开链接后会自动应用你的设置

### 技术实现

设置参数以 Base64 编码的 JSON 格式存储在 URL hash 中：`#prefs=<base64-encoded-json>`。当他人打开包含设置的分享链接时，偏好会自动解码并应用。

## 重置设置

点击设置面板底部的「重置」按钮可以将所有设置恢复为默认值。

## 设置持久化

所有设置会自动保存到浏览器的本地存储中，下次访问时会自动恢复。

## 技术细节

设置系统位于 `packages/core/src/preferences/` 目录：

- `types.ts` - 类型定义
- `defaults.ts` - 默认值
- `presets.ts` - 主题预设
- `storage.ts` - 存储管理
- `share.ts` - 分享功能
