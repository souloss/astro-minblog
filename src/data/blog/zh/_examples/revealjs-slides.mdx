Reveal.js 是一个强大的 HTML 演示框架。通过 `RevealSlides` 组件，你可以直接在博客文章中嵌入交互式幻灯片——无需任何外部工具。

## 使用说明与属性

`RevealSlides` 组件支持以下属性：

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `slides` | `string` | （必填） | 包含 `<section>` 元素的 HTML 字符串，每个 `<section>` 代表一张幻灯片 |
| `theme` | `string` | `white` / `night` | Reveal.js 主题名称（自动随博客亮/暗模式切换） |
| `width` | `number` | `960` | 演示文稿宽度（像素） |
| `height` | `number` | `540` | 演示文稿高度（像素） |

每个 `<section>` 元素代表一张幻灯片。支持嵌套 `<section>` 以实现垂直导航。组件会在博客主题切换时自动适配亮色/暗色主题。

### 幻灯片内容提示

- 使用标准 HTML 标签：`<h2>`、`<p>`、`<ul>`、`<pre><code>` 等
- 在 `<code>` 块上添加 `data-trim` 可自动去除首尾空白
- 使用 `<img>` 标签嵌入图片
- 使用 `class="fragment"` 支持 Reveal.js 的渐显效果
