Two.js is a renderer-agnostic 2D drawing API. The `TwoAnimation` component lets you create animated shapes using a simple JSON configuration, with optional autoplay for continuous motion.

## Spinning Shapes

Shapes with rotation animations applied. The `animate` property drives continuous rotation over a specified duration.

<TwoAnimation
  config='{"width":400,"height":300,"autoplay":true,"shapes":[{"type":"rectangle","x":100,"y":150,"width":80,"height":80,"fill":true,"fillColor":"#3b82f6","stroke":"#1e40af","animate":{"rotation":360,"duration":3000}},{"type":"rectangle","x":200,"y":150,"width":60,"height":60,"fill":true,"fillColor":"#8b5cf6","stroke":"#5b21b6","animate":{"rotation":360,"duration":2000}},{"type":"rectangle","x":300,"y":150,"width":100,"height":40,"fill":true,"fillColor":"#ec4899","stroke":"#9d174d","animate":{"rotation":360,"duration":4000}}]}'
/>

Each rectangle spins at a different speed, creating a mesmerizing asynchronous rhythm. The `duration` value is in milliseconds — lower values spin faster.

## Configuration Reference

The `TwoAnimation` component takes a `config` prop — a JSON string with this structure:

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `width` | `number` | `400` | Canvas width |
| `height` | `number` | `300` | Canvas height |
| `autoplay` | `boolean` | `false` | Enable animation loop |
| `shapes` | `array` | `[]` | Array of shape definitions |

### Shape Properties

| Property | Type | Description |
|----------|------|-------------|
| `type` | `string` | `circle`, `rectangle`, `ellipse`, or `line` |
| `x`, `y` | `number` | Position (center for circle/ellipse/rectangle, start for line) |
| `r` | `number` | Radius (circle only) |
| `width`, `height` | `number` | Dimensions (rectangle/ellipse) or end point (line) |
| `fill` | `boolean` | Whether to fill the shape |
| `fillColor` | `string` | Fill color |
| `stroke` | `string` | Stroke color |
| `animate` | `object` | Animation config: `{ rotation, duration }` |

### Animation

When `autoplay` is `true`, shapes with an `animate` property will continuously rotate. The `rotation` field is in degrees and `duration` is the loop time in milliseconds.
