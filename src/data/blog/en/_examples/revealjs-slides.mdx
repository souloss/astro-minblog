Reveal.js is a powerful HTML presentation framework. With the `RevealSlides` component, you can embed interactive slide decks directly in your blog posts — no external tools needed.

## Usage & Props

The `RevealSlides` component accepts the following props:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `slides` | `string` | (required) | HTML string containing `<section>` elements for each slide |
| `theme` | `string` | `white` / `night` | Reveal.js theme name (auto-switches for light/dark mode) |
| `width` | `number` | `960` | Presentation width in pixels |
| `height` | `number` | `540` | Presentation height in pixels |

Each `<section>` element represents one slide. You can nest `<section>` tags for vertical slide navigation. The component automatically switches between light and dark themes when the blog theme changes.

### Slide Content Tips

- Use standard HTML tags: `<h2>`, `<p>`, `<ul>`, `<pre><code>`, etc.
- Add `data-trim` to `<code>` blocks to auto-trim whitespace
- Embed images with `<img>` tags
- Slides support Reveal.js fragments with `class="fragment"`
